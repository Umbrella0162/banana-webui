# next-banana 🍌 项目交付文档

## 项目概览

**next-banana** 是一个基于 Next.js 16 的现代化 Gemini 图像生成应用。它完全在客户端运行，直接调用 Google Gemini API，无需后端服务器，非常适合部署在 Vercel 上。

### ✨ 核心特性

- **🍌 Banana Joy 主题**：清新愉悦的黄色主题设计。
- **⚡ 极速响应**：基于 Next.js 16 和 React 19，无服务器中转延迟。
- **🔒 隐私安全**：API 密钥仅存储在浏览器本地 (localStorage)，不上传服务器。
- **🎨 全功能支持**：
  - 支持 **Gemini 2.5 Flash**, **Gemini 3 Pro**, **Nano Banana Pro** 模型。
  - 文本生成图像 & 图像编辑（支持多图上传）。
  - 完整的参数控制：长宽比、分辨率、响应模式。
  - **Google Search** 工具集成。
  - 批量生成（1-4 张）。
- **📱 响应式设计**：完美适配桌面、平板和移动设备。

## 🚀 快速开始

### 本地开发

1. **安装依赖**
   ```bash
   npm install
   ```

2. **启动开发服务器**
   ```bash
   npm run dev
   ```

3. **访问应用**
   打开浏览器访问 [http://localhost:3000](http://localhost:3000)

### 部署到 Vercel

1. 将代码推送到 GitHub 仓库。
2. 在 Vercel 控制台导入该仓库。
3. **无需配置环境变量**（API Key 由用户在前端输入）。
4. 点击 **Deploy**。

## ⚠️ 注意事项

- **API Key**：用户需要自己的 Gemini API Key。应用首次加载时会提示输入。
- **模型权限**：请确保 API Key 关联的项目有权访问所选模型（如 Gemini 3 Pro）。
- **浏览器兼容性**：推荐使用最新版本的 Chrome, Edge, Firefox 或 Safari。
